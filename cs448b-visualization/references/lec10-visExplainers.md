# Lecture 10: Visual Explainers

## Core Concept
**Narrative Visualization**: Blending communication via imagery and text with interaction techniques.

## Storytelling Fundamentals

### Journalism Structure
| Element | Purpose |
|---------|---------|
| **Lede** | Anecdotal lead (1st paragraph) - grabs attention |
| **Nut Graf** | Core thesis paragraph - states the point |

### Visual Attention Direction
- **Montage/Editing**: Prescribed order, explicit direction, little viewer agency
- **Ensemble Staging**: Less prescribed, salience nudges attention, more viewer agency

## Seven Genres [Segel & Heer 2010]

| Genre | Characteristics |
|-------|-----------------|
| **Magazine Style** | Text + embedded visualizations |
| **Annotated Chart** | Chart with callouts explaining key points |
| **Partitioned Poster** | Multiple sections with visual hierarchy (beginning → middle → end → epilogue) |
| **Flow Chart** | Sequential explanation of process |
| **Comic Strip** | Panel-by-panel narrative progression |
| **Slide Show** | Discrete steps with navigation |
| **Film/Video/Animation** | Time-based, continuous narrative |

## Author-Driven vs Reader-Driven Spectrum

```
Author-Driven ←――――――――――――――――――→ Reader-Driven

Interactive    Martini Glass      Drill Down
Slideshow
```

### Author-Driven
- Prescribed ordering
- Stronger messaging
- Limited interactivity

### Reader-Driven
- Multiple orderings
- Less messaging
- More open interactivity

### Martini Glass Structure
1. **Stem**: Author-driven intro narrative
2. **Bowl**: Opens to reader exploration

## Engagement Techniques

### Draw Your Expectations
- User sketches prediction before seeing real data
- Creates personal investment
- Reveals misconceptions

### Simulations
- Interactive models users can manipulate
- Parameters affect outcomes
- Builds intuition about complex systems

## Design Principles

### Visual Hierarchy in Posters
1. Grab attention with image/position
2. Match content with visual prominence
3. Reduce priority for supporting details

### Key Tools
- **Idyll**: Markup language for interactive articles
- **Scrollytelling**: Narrative triggered by scroll position

## Summary
1. Narrative vis blends **imagery, text, and interaction**
2. Genres provide templates for different story types
3. Balance **author guidance** vs **reader agency**
4. Engagement techniques (draw, simulate) increase investment
